# Advanced-sockets-pygame
RTS movement integrated with sockets (multiplayer).
When trying to run the program server_game.py and server.py must be run first.
The ip_address in network.py should be changed to the ip of the machine running the servers
Multiple instances of main.py can be runned.

####
The program was done under a lot of time constraints for A level project so it might look a bit rough.
